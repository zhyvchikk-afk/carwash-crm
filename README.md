# CarWash CRM

CRM система для автомийки, розроблена на Django REST Framework + React.

## Технології

- Python
- Django
- Django REST Framework
- PostgreSQL
- React
- Vite
- CSS Modules
- JWT Authentication
- Telegram Bot API (aiogram)

## Основний функціонал

- Авторизація та реєстрація
- Управління автомобілями
- Онлайн-запис на автомийку
- Календар бронювань
- Адмін-панель зі статистикою
- Телеграм-бот для повідомлень
- Сповіщення адміну та клієнтам
